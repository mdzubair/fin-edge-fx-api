# Fin Edge API

A secure and scalable REST API for the Fin Edge Stock Management System. This application provides user authentication, stock management, portfolio tracking, withdrawal requests, ticket support, and admin management functionalities.

## Features

### Authentication
- User Registration
- User Login
- JWT Access Token Authentication
- Refresh Token Authentication
- Logout Functionality
- Protected Routes

### User Management
- User Profile Management
- Update Profile
- Change Password
- Role-Based Access Control

### Stock Management
- Add Stocks
- Update Stocks
- Delete Stocks
- View Stock Listings
- Stock Categories

### Portfolio Management
- Buy Stocks
- Sell Stocks
- Portfolio Tracking
- Transaction History

### Withdrawal Management
- Create Withdrawal Requests
- Withdrawal Approval Workflow
- Withdrawal Status Tracking

### Support Ticket System
- Create Support Tickets
- View Ticket History
- Admin Ticket Response
- Ticket Status Management

### Admin Dashboard
- User Management
- Stock Management
- Withdrawal Management
- Ticket Management
- Analytics Overview
